Para ejecutar el algoritmo de raytracing por favor compilar utilizando el Makefile que se encuentra en la carpeta bin del proyecto,
o bien en un entorno "terminal" Linux ejecutar los siguientes comandos desde la carpeta del proyecto:
	cd bin
	mingw32-make.exe

Eso debería compilar y abrir el programa.
Para ver el último resultado generado por el programa se puede abrir el ejecutable "main" que se encuentra también en la carpeta bin.
